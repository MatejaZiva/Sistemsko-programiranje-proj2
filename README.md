Zadatak 27:
Kreirati Web server koji klijentu omogućava pretragu umetničkih dela korišćenjem Europeana
API-a. Pretraga se može vršiti pomoću filtera koji se zadaju u okviru query-a. Spisak dela koja

korišćenjem GET metode. Ukoliko navedene informacije ne postoje, prikazati grešku korisniku.
